# Full-Stack Web Development

A repository focused on modern full-stack web development technologies, architectures, and implementation practices with [React](https://react.dev/), [Redux](https://redux.js.org/), [Node.js](https://nodejs.org/en), [TypeScript](https://www.typescriptlang.org/), [GraphQL](https://graphql.org/), and more.

## Overview

### Part 0: Fundamentals of Web Apps
**Technologies:** [HTML](https://developer.mozilla.org/en-US/docs/Web/HTML), CSS, JavaScript, HTTP

Introduction to web applications, HTTP requests, browser-server communication, and single-page applications (SPA). Includes basic HTML, CSS, and JavaScript fundamentals.

### Part 0123: Combined Exercises
**Technologies:** React, Node.js, Express, MongoDB

A consolidated collection of exercises from Parts 0-3, including the phonebook application that demonstrates full-stack development with React frontend and Node.js/Express backend connected to MongoDB.

### Part 1: Introduction to React
**Technologies:** [React](https://react.dev/), JSX, JavaScript

Core React concepts including components, props, state management with hooks (useState), and event handling. Exercises cover course information display and the Unicafe feedback application.

### Part 2: Communicating with Server
**Technologies:** React, [Node.js](https://nodejs.org/en), Axios, REST API, JSON

Building React applications that fetch and manipulate data from servers using REST APIs. Covers HTTP requests, promises, and rendering data collections.

### Part 3: Programming a Server with NodeJS and Express
**Technologies:** [Node.js](https://nodejs.org/en), Express, [MongoDB](https://www.mongodb.com/), Mongoose, REST API

Backend development with Node.js and Express framework. Creating RESTful APIs, connecting to [MongoDB](https://www.mongodb.com/docs/#welcome-to-the-mongodb-docs) databases, data validation, and deploying to production (Heroku).

### Part 4: Testing Express Servers, User Administration
**Technologies:** Node.js, Express, MongoDB, Jest, Supertest, [JWT](https://www.jwt.io/), Bcrypt

Advanced backend development including user authentication with [JSON Web Tokens (JWT)](https://www.jwt.io/), password hashing, testing with Jest, and implementing blog list application with user management.

### Part 5: Testing React Apps
**Technologies:** React, [Cypress](https://www.cypress.io/), Jest, React Testing Library

Frontend testing strategies including unit tests, integration tests, and end-to-end testing with [Cypress](https://www.cypress.io/). Covers test-driven development and browser automation.

### Part 6: State Management with Redux
**Technologies:** React, [Redux](https://redux.js.org/), Redux Thunk, React-Redux

Advanced state management using [Redux](https://redux.js.org/) for predictable and maintainable global state. Includes Redux store configuration, actions, reducers, and connecting Redux to React applications.

### Part 7: React Router, Custom Hooks, Styling
**Technologies:** React, React Router, Webpack, CSS, Custom Hooks

Advanced React patterns including client-side routing with React Router, creating custom hooks, styling applications with CSS-in-JS solutions, and configuring Webpack.

### Part 8: GraphQL
**Technologies:** GraphQL, Apollo Server, Apollo Client, [MongoDB](https://www.mongodb.com/)

Introduction to GraphQL for efficient data fetching. Implementing GraphQL servers with Apollo Server and integrating GraphQL clients in React applications with Apollo Client.

### Part 9: TypeScript
**Technologies:** [TypeScript](https://www.typescriptlang.org/), Node.js, React, Express

Type-safe development with [TypeScript](https://www.typescriptlang.org/). Covers TypeScript fundamentals, typing Node.js backends, and building type-safe React applications.

### Part 10: React Native
**Technologies:** [React Native](https://reactnative.dev/), Expo, Apollo Client, GraphQL

Mobile application development with [React Native](https://reactnative.dev/). Building cross-platform mobile apps, handling forms, routing, and integrating with GraphQL APIs.

### Part 11: CI/CD
**Technologies:** GitHub Actions, Testing, Deployment, Docker

Continuous Integration and Continuous Deployment pipelines. Automated testing, linting, building, and deploying applications using GitHub Actions and other CI/CD tools.

### Part 12: Containers
**Technologies:** Docker, Dockerfile, Docker Compose, Containerization

Containerization with Docker. Creating Dockerfiles, orchestrating multi-container applications with Docker Compose, and deploying containerized applications.

## 🔧 Technologies & Tools

This repository provides an overview of a modern web development technology stack:

**Frontend:**
- [React](https://react.dev/) - The library for web and native user interfaces
- [Redux](https://redux.js.org/) - A JS library for predictable and maintainable global state management
- [React Native](https://reactnative.dev/) - Framework for building native mobile applications
- [HTML](https://developer.mozilla.org/en-US/docs/Web/HTML) - HyperText Markup Language
- CSS - Styling and layouts
- JSX - JavaScript XML syntax

**Backend:**
- [Node.js](https://nodejs.org/en) - Open-source, cross-platform JavaScript runtime environment
- Express - Web application framework for Node.js
- GraphQL - Query language for APIs
- REST API - Architectural style for web services

**Databases:**
- [MongoDB](https://www.mongodb.com/) - NoSQL document database ([Documentation](https://www.mongodb.com/docs/#welcome-to-the-mongodb-docs))
- Mongoose - MongoDB object modeling for Node.js
- [PostgreSQL](https://www.postgresql.org/) - Advanced open-source relational database
- Sequelize - Promise-based Node.js ORM for SQL databases
- Redis - In-memory data structure store

**Programming Languages:**
- JavaScript - Core programming language
- [TypeScript](https://www.typescriptlang.org/) - JavaScript with syntax for types

**Testing:**
- [Cypress](https://www.cypress.io/) - Browser testing framework
- Jest - JavaScript testing framework
- Supertest - HTTP assertion library
- React Testing Library - Testing utilities for React

**Authentication & Security:**
- [JWT (JSON Web Tokens)](https://www.jwt.io/) - Secure token-based authentication
- Bcrypt - Password hashing

**DevOps & Tools:**
- Docker - Containerization platform
- GitHub Actions - CI/CD automation
- Webpack - Module bundler
- Babel - JavaScript compiler
